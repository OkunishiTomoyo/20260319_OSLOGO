# UMIDAS デザインコンセプト設計書

## 1. コンセプトワード策定

### 1.1 コアマインドからの導出

```
代表者のコアマインド
    承認欲求 → 自己実現
    「こいつが作ったんだよ」と言われたい
        ↓
会社としての落とし込み
    心に余裕を生み出す → チャレンジできる状態を作る
    誠実に・自然体で向き合う
        ↓
デザインコンセプト（ロゴが語るべき物語）
```

### 1.2 デザインコンセプトワード候補

#### 【推奨】案A:「静かに満ちて、溢れ出す」

- 企業理念「心に余裕を生み出す」の視覚的翻訳
- 水色→青のグラデーション＝満たされていく過程
- 溢れ出す＝余裕から生まれるチャレンジ・成功
- 「青い熱狂」＝静かだが完全燃焼しているイメージと合致

#### 案B:「誠実が、波紋になる」

- 一つの誠実な行動が波紋のように広がる
- コスト削減→余裕→チャレンジ→成功の連鎖
- 旧社名オープンセレスの波紋ロゴとの精神的継承

#### 案C:「自然体で、生み出す」

- 代表者の最も強い行動原理「自然体」
- 力まず、ありのままでいることが最大の生産性
- 水の自然な流れ＝無理のない誠実さ

### 1.3 推奨コンセプト

> **「静かに満ちて、溢れ出す」**

理由：
- 水色〜青のカラー指定と自然に結びつく
- 「生み出す（UMIDAS）」の語感＝海（UMI）＋出す（DASU）を内包
- 上野氏の「青い熱狂」への共感と一致
- 顧客層（金融・士業）に響く落ち着きと信頼感
- 「溢れ出す」が成長・成功のメタファーとして機能

## 2. ネーミング分析「UMIDAS」

### 2.1 言葉の多層構造

```
UMIDAS
├── うみだす（生み出す）＝ 創造する、新しい価値を作る
├── UMI（海）＋ DAS（出す）＝ 海から出す → 海の恵み
├── UMI（産み）＋ DAS ＝ 産み出す → 誕生・孵化
└── 海 → 海ガメ（ホヌ）→ 幸せを運ぶ
```

### 2.2 ロゴに活かせるダブルミーニング

| 読み | 意味 | デザインへの翻訳 |
|------|------|----------------|
| 生み出す | 創造・価値創出 | 溢れ出すモーション |
| 海+出す | 海の恵み・広がり | 水・波のモチーフ |
| 産み出す | 誕生・成長 | 卵→海ガメの成長 |

## 3. カラーシステム

### 3.1 プライマリーパレット

```
Light Blue (優しさ・思いやり)
    #7EC8E3  ← メインの水色
    透明感があり、信頼と柔らかさを同時に表現

Deep Blue (誠実・信頼)
    #2B6CB0  ← グラデーションの到達点
    バキバキしすぎない、温かみのある深い青

Mid Blue (つなぎ・成長)
    #4A9BD9  ← グラデーションの中間
    水色から深い青への自然な遷移
```

### 3.2 グラデーション定義

```
メイングラデーション:
    #7EC8E3 → #4A9BD9 → #2B6CB0
    方向: 左下→右上（上昇・成長のイメージ）
    または: 中心→外側（溢れ出すイメージ）
```

### 3.3 サブカラー

```
White / Off-White:
    #FFFFFF / #F8FAFB
    背景・余白用

Dark (テキスト用):
    #1A365D
    ネイビー寄りの暗色（真っ黒は使わない）
```

### 3.4 NGカラー

- 緑系全般
- 赤（メラ社との差別化、金融系顧客にはアグレッシブすぎる）
- 紫（方向性が異なる）
- バキバキした原色の青（#0000FF系）

## 4. タイポグラフィ方向性

### 4.1 ロゴタイプ書体の方向性

| 方向性 | 説明 | 参考イメージ |
|--------|------|------------|
| モダンサンセリフ | 洗練・信頼感。コックピット系 | Futura, Montserrat系 |
| ジオメトリック | 幾何学的で知的。トライジン系 | Poppins, Quicksand系 |
| カスタムレタリング | 文字自体をロゴ化。遊び心あり | オリジナル設計 |

### 4.2 書体選定の基準

- **読める**こと（ヨウジヤマモトほど崩さない）
- **遊び心**があること（シンプルすぎない）
- **6文字の横幅**を考慮したバランス
- **金融・士業の顧客**に受け入れられる品格

### 4.3 推奨アプローチ

「UMIDAS」のアルファベットをベースに、一部の文字にモチーフ（水滴・波・海ガメの甲羅パターン等）を忍ばせるセミカスタム書体。

## 5. ロゴ展開想定

### 5.1 使用シーン

| シーン | サイズ | 背景 | 考慮事項 |
|--------|--------|------|---------|
| 名刺 | 小 | 白/淡色 | 渡す瞬間のインパクト、指との関係性 |
| Tシャツ | 大 | 黒/白 | チーム全員着用、営業アポで着る |
| Web/SNS | 中 | 各種 | アイコン化できるか |
| 資料ヘッダー | 小〜中 | 白 | 法人向け資料の信頼感 |
| 看板/サイン | 大 | 各種 | 遠距離視認性 |

### 5.2 ロゴバリエーション要件

| バリエーション | 用途 |
|--------------|------|
| フルロゴ（マーク+テキスト・横組み） | 標準使用 |
| フルロゴ（マーク+テキスト・縦組み） | 縦長スペース |
| マークのみ | アイコン、ファビコン |
| テキストのみ | テキスト中での使用 |
| 単色（白） | 暗い背景 |
| 単色（ダーク） | 明るい背景でカラー不可時 |

## 6. 競合・参照分析

### 6.1 上野氏が好むデザインの共通点

```
コックピット + アクロシールズ + トライジン
    ↓ 共通要素の抽出
    ・シャープさと洗練
    ・やや固め（カジュアルすぎない）
    ・マークに意味が込められている
    ・テキストはクリーンで読みやすい
    ・配色は抑制的（2〜3色以内）
```

### 6.2 避けるべきデザイン領域

```
ハワイアンショップ系 ← 海ガメ出しすぎ
女性向けエステ系 ← 柔らかすぎ
スタートアップ系テック ← 無機質すぎ
手書き/オーガニック系 ← カジュアルすぎ
```
