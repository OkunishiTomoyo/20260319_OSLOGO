# UMIDAS ロゴコンセプト提案書

## 全6案概要

デザインコンセプト「**静かに満ちて、溢れ出す**」を軸に、6つのロゴ案を提案します。

---

## A案: 水紋 ×ロゴタイプ — 「波紋の先へ」

### コンセプト
一つの誠実な行動（水滴）が波紋となって広がり、やがて大きな変化を生み出す。

### ビジュアル
```
    ○ ← 水滴が落ちた瞬間
   ╱ ╲
  (   ) ← 波紋（1〜2重の同心円弧）
 ╱     ╲
───────────

  U M I D A S
```

- シンボルマーク: 水滴＋波紋の同心円弧（上半分のみ）
- 波紋は2〜3本。最外周が途切れていて「まだ広がり続ける」ことを暗示
- カラー: 中心（#7EC8E3 薄い水色）→ 外周（#2B6CB0 深い青）グラデーション
- テキスト: クリーンなジオメトリックサンセリフ

### 意味
- 水滴 = 一つの誠実な行動
- 波紋 = その影響が広がっていく
- 途切れた最外周 = まだ成長途中、可能性は無限

### 名刺展開
- 名刺の角から波紋が広がるデザイン
- 渡す瞬間、相手の手に波紋が「伝わる」演出

### 適性
- 法人向け資料: ★★★★★（信頼感抜群）
- Tシャツ: ★★★☆☆（シンプルで使いやすい）
- アイコン化: ★★★★☆（水滴+波紋で明快）

## B案: 海流ロゴタイプ — 「流れるUMIDAS」

### コンセプト
UMIDASの文字自体を流れる水のように設計。文字が水流となり、読み進めるうちに「溢れ出す」動きを感じる。

### ビジュアル
```
  ┌─ 水流の起点（細い）
  │
  U ╱M ─ I ─ D ─ A ─ S
                        ╲── 水流の終点（太く溢れる）
```

- 「U」の左下から水が流れ始め、「S」で溢れ出すように太くなる
- 文字の線幅が左から右へ微妙に太くなるグラデーション
- 「D」の縦棒を水滴に見立てるなどの遊び心
- カラー: 左（#7EC8E3）→ 右（#2B6CB0）のグラデーション

### 意味
- 文字を読む左→右の動きが、成長・充足のプロセスそのもの
- 細い→太い = 余裕が生まれて溢れ出す
- 「U」の形 = 器（心の余裕の器）

### 名刺展開
- 文字の流れを名刺全面に活かしたレイアウト
- 裏面に「S」から溢れ出した水流が広がるパターン

### 適性
- 法人向け資料: ★★★★☆（洗練されつつ個性的）
- Tシャツ: ★★★★★（文字だけで映える）
- アイコン化: ★★★☆☆（「U」単体をアイコンに）

## C案: 甲羅六角 — 「ホヌの祝福」

### コンセプト
海ガメ（ホヌ）の甲羅パターンを抽象化した六角形モチーフ。幸せを運ぶ海ガメの甲羅が、見る人を守るシールドとなる。

### ビジュアル
```
     ╱╲
    ╱  ╲
   │ ╱╲ │  ← 甲羅の六角形パターン
   │╱  ╲│     （抽象化して2〜3の六角形の集合）
    ╲  ╱
     ╲╱

  U M I D A S
```

- シンボルマーク: 六角形の集合体（海ガメの甲羅を抽象化）
- 中央の六角形が最も明るく、外側に行くほど深い青に
- 一見すると幾何学模様、よく見ると海ガメの甲羅がモチーフだと分かる
- テキスト: シャープなモダンサンセリフ

### 意味
- 六角形 = 自然界で最も効率的な構造（ハニカム）= コスト削減
- 甲羅 = 守る・包む = 心の余裕を守る
- ホヌ = 幸せを運ぶ
- パッと見では分からない = 「見えない価値」を象徴

### 名刺展開
- 甲羅パターンをエンボス（空押し）加工で名刺表面に
- 触ると分かる＝見えない価値が触れると分かる

### 適性
- 法人向け資料: ★★★★★（知的・洗練・信頼）
- Tシャツ: ★★★★☆（幾何学模様として映える）
- アイコン化: ★★★★★（六角形で明快）

## D案: ライジングドロップ — 「青い熱狂」

### コンセプト
水滴が上昇する瞬間を捉えたマーク。水でありながら炎のように上へ向かう。「青い熱狂」を形にする。

### ビジュアル
```
      ╱╲
     ╱  ╲    ← 上昇する水滴（炎のようなフォルム）
    │    │
    │    │
     ╲  ╱
      ──

  U M I D A S
```

- シンボルマーク: 逆さの水滴＝上に向かって立ち上る形
- 水滴のフォルムだが、炎のようにも見えるダブルミーニング
- 内部にグラデーション: 下部（#7EC8E3 水色）→ 上部（#2B6CB0 深い青）
- 上部の先端がわずかに揺らいで、静かな熱を感じさせる

### 意味
- 水滴+炎 = 青い熱狂（静かだが完全燃焼）
- 上昇 = 成長・成功への道
- 水色→深い青 = 満たされていく過程
- 「自然体だが、燃えている」という矛盾の共存

### 名刺展開
- 名刺下部から上に向かってドロップが立ち上るデザイン
- ロゴと「心に余裕を生み出す」のコピーを配置

### 適性
- 法人向け資料: ★★★★☆（個性的だが品格あり）
- Tシャツ: ★★★★★（胸元に映える・インパクト大）
- アイコン化: ★★★★★（ドロップ単体で明快）

## E案: オーバーフロー — 「溢れ出す器」

### コンセプト
「U」の文字を器に見立て、そこから水が溢れ出す。UMIDASの頭文字であり、「心に余裕を生み出す」を直接表現。

### ビジュアル
```
  ～～～～～～
  │ ≋≋≋≋≋│ ← 「U」の器から水が溢れている
  │      │
   ╲    ╱
    ╲  ╱
     ──
       MIDAS
```

- 「U」を器として大きく配置、そこから水が溢れ出す表現
- 「MIDAS」は「U」の右側にやや小さく配置
- 溢れ出す水のラインは有機的な曲線
- Midasはギリシャ神話の「触れるものすべてを黄金に変えるミダス王」のダブルミーニング

### 意味
- U = 器 = 心の余裕の容器
- 溢れ出す水 = 余裕が満ちて自然と溢れる状態
- MIDAS = 触れるものに価値を与える王（コスト削減で価値を生む）
- U + MIDAS = UMIDAS（生み出す）

### 名刺展開
- 名刺の上辺から水が溢れ出すデザイン
- 「生み出す」感覚を手に取った瞬間に感じる

### 適性
- 法人向け資料: ★★★☆☆（やや遊びすぎか要調整）
- Tシャツ: ★★★★☆（ユニークで会話のきっかけに）
- アイコン化: ★★★★★（U+水のコンビネーション）

## F案: コネクトライン — 「つなぐ、自然体で」

### コンセプト
UMIDASの6文字を一筆書きのように1本の線でつなぐ。切れ目なく続く線＝誠実に途切れなく向き合い続ける姿勢。

### ビジュアル
```
  U─M─I─D─A─S
  └──────────┘ ← 一本の線がすべてをつなぐ
```

- 6文字を1本の流れるラインで構成（完全な一筆書きでなくてもよいが、連続性を感じる）
- 線の太さに微妙な強弱（呼吸するようなリズム）
- 線のカラー: 水色〜青のグラデーション
- ミニマルだが、線の流れに職人的な美しさ

### 意味
- 一本の線 = 途切れない誠実さ
- 文字のつながり = 人と人のつながり
- 強弱のある線 = 自然体（力まない）
- 始まりと終わりがつながる = 循環・持続

### 名刺展開
- 名刺の縁を1本の線が巡るデザイン
- ロゴの線と名刺の枠線が一体化

### 適性
- 法人向け資料: ★★★★★（最も洗練・信頼感）
- Tシャツ: ★★★☆☆（シンプルすぎる可能性）
- アイコン化: ★★☆☆☆（文字依存のため難しい）

## 推奨案

### 第1推奨: C案「甲羅六角」+ B案「海流ロゴタイプ」のハイブリッド

- **シンボルマーク**: C案の六角形（甲羅の抽象化）
- **ロゴタイプ**: B案の流れるUMIDAS文字
- **理由**:
  - 社内文化（海ガメ＝幹部、卵＝新人）との深い結びつき
  - コックピット社的な「マーク+テキスト」フォーマット
  - 六角形はアイコンとして最も汎用性が高い
  - 「一見幾何学、よく見ると海ガメ」の二重構造が上野氏の好む意味の深さと合致

### 第2推奨: D案「ライジングドロップ」

- 「青い熱狂」という上野氏自身が共感したワードの直接表現
- Tシャツ映え最強
- 水と炎のダブルミーニングが斬新

### 第3推奨: A案「水紋」

- 最も王道で信頼感がある
- 金融・士業顧客への安心感
- 旧社名ロゴ（波紋）からの精神的継承
