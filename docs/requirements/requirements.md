## 4. ロゴデザイン要件

### 4.1 制作物

| 制作物 | 用途 | 優先度 |
|--------|------|--------|
| サービスロゴ / 会社ロゴ | UMIDAS全体のシンボル | **必須** |
| 名刺デザイン | 営業時の第一印象 | 将来的に |
| Tシャツデザイン | チームウェア（営業アポ時にも着用） | 将来的に |

### 4.2 ロゴ形式の方向性（2パターン検討）

#### パターンA: シンボルマーク + 社名テキスト

- コックピット社のように、象徴的なマーク + シンプルなテキスト
- マークに意味を込めやすい
- 名刺やTシャツへの展開がしやすい

#### パターンB: 文字自体をロゴ化（ロゴタイプ）

- UMIDAS（またはカタカナ「ウミダス」/ ひらがな「うみだす」）の文字自体をデザイン
- トライジン社のように文字がそのままデザインになるイメージ
- 注意: 「生み出す」「UMIDAS」は6文字で横幅を取る
- シンプルすぎないよう、オシャレな書体にしたい
- ヨウジヤマモトほどやりすぎない（読めなくなるレベルはNG）

**→ 両パターン提案し、しっくりくる方を採用**

### 4.3 カラー

| 項目 | 指定 |
|------|------|
| メインカラー | **水色〜青**（light blue〜blue） |
| トーン | 優しさ・柔らかさのある水色がベース |
| グラデーション | 薄い水色 → 濃い青のグラデーションも可 |
| 濃い青の方向性 | バキバキした濃い青ではなく、**柔らかみのある濃い青** |
| NG色 | 緑（「緑はあんまりいらない」と明言） |

#### カラーが表現すべき意味

| 色 | 表現する価値 |
|------|------|
| 水色 | 優しさ、思いやり、柔らかさ |
| 青 | 誠実さ、信頼 |
| グラデーション | 満たされて溢れ出る状態、成長 |

### 4.4 デザインテイスト

| 方向性 | 詳細 |
|---------|------|
| 全体感 | **柔らかすぎず、やや固め** — コックピット社やアクロシールズ社のような洗練されたイメージ |
| 雰囲気 | 信頼感・誠実さが伝わるが、冷たくない |
| 顧客印象 | 金融系・士業の信頼重視の顧客に刺さるデザイン |
| 遊び心 | 少し遊び心を入れたい（ただしやりすぎない） |
| NG | ハワイのショップ感、過度にカジュアル、読めないほど崩した書体 |

### 4.5 モチーフ候補

録音から抽出されたモチーフアイデア：

| モチーフ | 意味・背景 | 優先度 |
|---------|----------|--------|
| **水・波紋** | 溢れ出す、波及する、広がっていく | 高 |
| **海ガメ（ホヌ）** | ハワイ語で幸せを運ぶ生き物。社内で幹部＝海ガメ、新人＝卵の比喩を使用 | 中 |
| **青い炎** | 完全燃焼の象徴。「青い熱狂」＝静かだが赤より燃えている | 中（斬新だが要検証） |
| **溢れ出すイメージ** | 心の余裕が満たされて溢れ出る状態 | 高 |
| **卵→海ガメ** | 成長・孵化のメタファー | 低（サブモチーフ） |

### 4.6 デザインコンセプト（言語化の方向性）

ロゴの根底に置くべきコンセプトワード（「○○を生み出す」の形）：

- 企業理念: **心に余裕を生み出す**
- デザインコンセプト候補（セッションで詰める領域）:
  - 「誠実の○○」系
  - 「○○を生み出す」系
  - 溢れ出す・満たされる系

**→ デザインコンセプトワードを確定させてからデザインに落とし込む流れ**

## 7. 重要な制約・注意事項

### 7.1 絶対条件

- **意味合いを最重要視**: 「デザインでいい感じにする」のではなく、意味を込める
- **誠実さが伝わる**: 顧客（金融系・士業）に信頼感を与える
- **名刺・Tシャツへの展開を想定**: 単体で完結せず、活用シーンを考慮

### 7.2 避けるべきもの

| NG項目 | 理由 |
|--------|------|
| 緑色の使用 | 代表者が「緑はいらない」と明言 |
| ハワイショップ感 | 海ガメモチーフを出しすぎると陥る |
| 読めないほど崩した書体 | 認知性の低下 |
| 過度に柔らかい/女性的なデザイン | 代表者のイメージに合わない |
| バキバキした濃い青 | 柔らかさが失われる |

### 7.3 デザイン上の判断ポイント

- 「UMIDAS」のロゴなのか「生み出す」のロゴなのか → 確認必要
- ホールディングスロゴとの関係性 → 将来的な体系を考慮
- 海ガメモチーフの採用度合い → がっつり vs さりげなく vs 不採用
