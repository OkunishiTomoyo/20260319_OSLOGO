UMIDAS LOGO DESIGN

ロゴデザイン提案 — 全20パターン

Design Concept | 静かに満ちて、溢れ出す
PATTERN 01 水紋 — Ripple

「波紋の先へ」

水滴が落ちた瞬間に広がる波紋をモチーフに。一つの誠実な行動が大きな変化を生み出す。最外周が途切れていることで「まだ広がり続ける」可能性を暗示。旧社名ロゴ（波紋）の精神的継承。

シンボル+テキスト 信頼感 王道 金融・士業向け
PATTERN 02 甲羅六角 — Hexagon Shell

「ホヌの祝福」

海ガメの甲羅パターンを六角形で抽象化。一見幾何学模様、よく見ると海ガメ。社内文化（幹部＝海ガメ、新人＝卵）と深く結びつく。六角形はハニカム構造＝効率性＝コスト削減も暗示。

推奨 シンボル+テキスト 知的 海ガメ アイコン化◎
PATTERN 03 ライジングドロップ — Rising Drop

「青い熱狂」

水滴が上昇する瞬間。水でありながら炎のように上へ向かう。静かだが完全燃焼する「青い熱狂」を形にした。上野氏がセッション中に共感した概念の直接表現。

推奨 シンボル+テキスト 水×炎 斬新 Tシャツ映え
PATTERN 04 オーバーフロー — Overflow U

「溢れ出す器」

UMIDASの頭文字「U」を器に見立て、そこから水が溢れ出す。「心に余裕を生み出す」の直接的ビジュアル化。MIDASはギリシャ神話の「触れるもの全てを価値に変えるミダス王」のダブルミーニング。

ロゴタイプ融合 ストーリー性 ダブルミーニング
PATTERN 05 フロータイプ — Flow Type

「流れるUMIDAS」

文字の太さが左（U=細い）から右（S=太い）へ徐々に変化するグラデーション書体。読み進めると「溢れ出す」感覚。Sの右から水滴が飛び出すアクセント。文字自体がデザインになるトライジン型。

ロゴタイプ トライジン型 Tシャツ映え
PATTERN 06 タートル抽象 — Abstract Honu

「海を渡るホヌ」

海ガメを線画で抽象化。甲羅は波のラインパターンで構成。直接的なイラストではなく、ラインアートとしてスタイリッシュに昇華。海ガメモチーフをしっかり出しつつ、ハワイ感を抑制。

シンボル+テキスト 海ガメ ラインアート
PATTERN 07 ウェーブサークル — Wave Circle

「溢れ出す海」

円形の器の中に海面を配置し、そこから水滴が飛び出す。「満たされて溢れ出す」状態の直接表現。円は調和・信頼の象徴。水滴が円の外に飛び出すことで、枠を超えた成長を表現。

シンボル+テキスト 溢れ出す アイコン化◎
PATTERN 08 モノグラム — Monogram U

「器としてのU」

「U」をスタイリッシュなモノグラムとして大きく配置し、シンボルマーク化。Uの器から左右に水が溢れ出すラインで動きを加える。コックピット社のようなマーク+テキストの王道構成。

推奨 モノグラム コックピット型 汎用性◎
PATTERN 09 ブルーフレイム — Blue Flame

「完全燃焼する誠実さ」

青い炎をモチーフに。赤い炎は不完全燃焼、青い炎は完全燃焼。静かに見えて実は最も高温＝静かだが最も本気。上野氏の「青い熱狂」セッションから直接インスパイア。内部のグラデーションで奥行きを表現。

シンボル+テキスト 青い炎 インパクト Tシャツ映え
PATTERN 10 ミニマルウェーブ — Minimal Wave

「波をまとう文字」

UMIDASの文字を横切る波ライン1本で、水のイメージを最小限に統合。最もミニマルなアプローチ。文字自体の力を信じ、装飾は波ラインのみ。名刺に最も映え、あらゆるサイズで破綻しない。

ロゴタイプ ミニマル 汎用性◎◎ 名刺向き

PATTERN 11–20 — Extended Collection

PATTERN 11 コンパス — Compass

「正しい方向へ、誠実に導く」

コンパスマークで「導く存在」を表現。クライアントを正しい方向へ導くナビゲーター。北の針が水色〜青のグラデーションで、上昇＝成功への方向を示す。コックピット社に近い固めのデザイン。

推奨 シンボル+テキスト 導く コックピット型 信頼感◎
PATTERN 12 シールドウェーブ — Shield Wave

「信頼で、守り抜く」

盾の中にU字と波を統合。信頼・守る・誠実の三要素を1つのマークに凝縮。金融・士業の顧客に最も刺さる「守りの信頼感」。盾の下半分が水で満たされ、心の余裕を表現。

推奨 シンボル+テキスト 信頼の盾 金融・士業◎ アクロシールズ型
PATTERN 13 ホライズン — Horizon

「水平線から昇る価値」

水平線と日の出のモチーフ。海面から新しい価値が生まれる瞬間を捉えた。UMIDAS（海+出す）の語源を最も直接的に表現。テキストを下部に大きく配置し、マークは控えめに上部で象徴する。

テキスト主体 UMI+DASU 新生 清涼感
PATTERN 14 スタックBOLD — Stacked Bold

「力強く、生み出す」

極太のUMIDASテキスト + 左のブルーアクセントライン + 水色〜青のアンダーバー。文字の力だけで勝負するミニマルかつ力強いデザイン。日本語「生み出す」を添えることで二言語の美しい対比を生む。

ロゴタイプ BOLD ミニマル Tシャツ映え◎
PATTERN 15 エッグハッチ — Egg Hatch

「殻を破り、生み出す」

卵が孵化する瞬間をマーク化。社内文化の「新人＝卵、幹部＝海ガメ」と直結。殻を破って飛び出す粒子は、生まれる価値・可能性の象徴。「産み出す」の意味を最も直接的に表現。

シンボル+テキスト 孵化 社内文化 成長
PATTERN 16 インフィニティ — Infinity Flow

「尽きない、生み出す力」

無限大（∞）を水流ラインで描いたマーク。持続的に価値を生み出し続ける循環を表現。∞は「心に余裕を生み出す」→「チャレンジ」→「成功」→「さらなる余裕」の無限ループ。

シンボル+テキスト 無限 循環 持続
PATTERN 17 ドロップレター — Droplet Letter

「一滴が、すべてを変える」

UMIDASの「D」を水滴形に置き換えたロゴタイプ。他の文字はクリーンなまま、Dだけが水モチーフに変わるさりげない遊び心。トライジン型の「文字の一部がデザインになる」アプローチ。

推奨 ロゴタイプ さりげない トライジン型 汎用性◎
PATTERN 18 レイヤードウェーブ — Layered Waves

「重なる波、深まる信頼」

3本の波ラインが薄い水色→中間→深い青と重なる。水色〜青のグラデーション指定を最も素直に表現。波の太さも徐々に太くなり、信頼が深まる様を表す。シンプルだが記憶に残る。

シンボル+テキスト 3層グラデ 波 アイコン化◎
PATTERN 19 ダイヤモンドドロップ — Diamond Drop

「誠実が結晶する」

ダイヤモンド形の中に水滴を配置。誠実さの積み重ねが結晶＝ダイヤモンドになるというメタファー。カット面の線が知的な印象を与え、水滴が内部で輝く。高級感と信頼のバランス。

シンボル+テキスト 結晶 高級感 知的
PATTERN 20 カレントタイプ — Current Type

「海流に乗る」

大きなUMIDAS文字の背景に海流カーブを配置。「I」のドットを水滴に変更し遊び心を加える。下部に波のアクセントラインと日本語「生み出す」。最もスケール感のある力強いデザイン。

推奨 ロゴタイプ スケール感 日本語併記 名刺◎
